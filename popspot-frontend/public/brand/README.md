# popspot 브랜드 에셋

popspot(팝스팟) — 서울 팝업스토어 발견 서비스 — 의 로고/색/폰트 가이드와 원본 파일 모음입니다.

---

## 1. 로고 컨셉 — "P-핀"

글자 **P** 의 카운터(속파임) 속 점 = 위치 점, 기둥 끝의 뾰족함 = 핀 꼬리.
즉 **P 이자 지도 핀(spot)** 이라는 이중 의미를 한 글자에 담은 모노그램입니다.
(popspot = **pop**up + **spot** → 등장하는 장소 → 핀)

단색 1색으로 성립하도록 설계해, 라임/잉크/흰색 어떤 배경·크기에서도 동작합니다.

---

## 2. 파일 목록

| 파일 | 용도 |
|---|---|
| `popspot-favicon.svg` | 앱 아이콘 / 파비콘 (라임 라운드 사각 + 흰 마크, 512×512, rx 116). `app/icon.svg` 와 동일 |
| `popspot-mark-ink.svg` | 마크 단독 — 밝은 배경 (잉크 `#18181B`) |
| `popspot-mark-lime.svg` | 마크 단독 — 라임 `#88C828` |
| `popspot-mark-white.svg` | 마크 단독 — 어두운 배경 (흰색) |
| `popspot-logo.svg` | 가로형 락업(마크+워드마크) — 밝은 배경 |
| `popspot-logo-white.svg` | 가로형 락업 — 어두운 배경 |

> **사이트 내부에서는** 정적 SVG 대신 React 컴포넌트
> [`src/components/layout/Logo.tsx`](../../src/components/layout/Logo.tsx) 를 쓰세요.
> 마크 + 워드마크를 함께 렌더링하며 크기/색을 props 로 제어합니다 (헤더·푸터·로그인이 이걸 사용).

---

## 3. 색 (팔레트)

| 이름 | HEX | 용도 |
|---|---|---|
| Lime 500 | `#88C828` | 메인 브랜드 / 마크 |
| Lime 400 | `#A8E645` | 어두운 배경용 마크·포인트 |
| Lime 300 | `#C2F970` | 강조·포인트(점) |
| Ink | `#18181B` | 본문/워드마크(라이트) |
| Cream 200 | `#F5F3EE` | 배경(라이트) |

---

## 4. 폰트

- **워드마크 / 본문 = Wanted Sans** (`"Wanted Sans Variable"`, jsdelivr CDN, weight 800 + 자간 -0.05em).
- 코드/숫자 = JetBrains Mono.
- `globals.css` 의 Tailwind `@theme` `--font-sans` / `--font-display` 가 사이트 전체를 지배합니다.

> ⚠️ `popspot-logo*.svg` 의 워드마크는 `<text>` + Wanted Sans 폰트 패밀리로 그려져 있습니다.
> Wanted Sans 가 설치/로드된 환경(웹사이트 포함)에서는 정확히 렌더되지만, 폰트가 없는 외부 도구에서
> 열면 시스템 폰트로 대체됩니다. **외부 배포용으로 글리프 고정이 필요하면 디자인 툴에서 텍스트를
> 아웃라인(path)으로 변환** 후 내보내세요.

---

## 5. 사용 규칙

- **여백(클리어스페이스):** 마크 높이의 최소 50% 를 로고 사방에 비워둡니다.
- **최소 크기:** 마크 단독 16px, 가로 락업 88px 이상.
- **하지 말 것:** 비율 왜곡 / 임의 색 변경 / 그림자·외곽선 추가 / 워드마크 글꼴 교체 / 마크와 워드마크 간격 임의 변경.
